HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta charset= utf-8 is a HTML tag that describes the web page's character encoding. It lets the browser know what kind of character encoding is used within the HTML file.
  • The "ul" and "ol" are two different ways to create unordered and ordered lists, and using the "li" child elements within them.
  • h1-h6 are header elements of varying sizes, and h1 is the biggest.
  • Section elements help organize content in a webpage.

Git

  • git status: checks what branche we are currently in.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git clone [url]: helps to retrieve repos via URL.
  • git status: shows the status of any modified files in a working directory.
  • git add [file]: Adds the existing file of how it looks currently to the next commit (stage).
  • git add -A: Stages all changes of all working files in the working bench.
  • git commit -m "[descriptive message]": a command that commits my staged content as a new commit snapshot.
  • git pull origin main: a command that syncs our local branch to the remote branch in Github. Helps keeps things up to date!The process of syncing and merging code into Github is how projects are continually updated with more features.
  • git push origin [branch name; example: feature/add-html]: Pushes all of our changes into the repo based on the branch.
